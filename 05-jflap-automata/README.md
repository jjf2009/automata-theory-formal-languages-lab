# 05 — JFLAP Finite Automata (DFA & NFA)

**Topic:** visual automata · JFLAP · DFA · NFA  
**Tool:** JFLAP 7.1  

## Files

| File | Role |
|------|------|
| `dfa.jff` | Deterministic finite automaton model |
| `nfa.jff` | Nondeterministic finite automaton model |

## How to open

```bash
java -jar ../tools/jflap/JFLAP7.1.jar
```

Then **File → Open** and select `dfa.jff` or `nfa.jff`.

## Why JFLAP?

JFLAP lets you draw states and transitions, test inputs visually, and compare DFA/NFA behavior without writing code—ideal alongside the C simulators in modules 02 and 03.
