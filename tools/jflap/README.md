# JFLAP 7.1 (third-party)

[JFLAP](https://www.jflap.org/) is a visual tool for teaching automata theory: DFA, NFA, regular expressions, CFG, PDA, Turing machines, and more.

## Run

```bash
java -jar JFLAP7.1.jar
```

Requires a Java Runtime Environment.
